# Chat Zoera 😈💬

Projeto de chat simples com sistema de mensagens e comandos zoeira.

## 🚀 Tecnologias
- HTML  
- CSS  
- JavaScript  
- (Futuramente) Banco de dados + WebSocket  

## 💡 Funcionalidades
✅ Enviar mensagens  
✅ Mostrar mensagens na tela  
✅ Comandos especiais (ex: /fig)

## 📌 Como usar
1. Baixe o projeto  
2. Abra o arquivo `index.html` no navegador  
3. Comece a usar o chat
